## Доступные команды:


```
Команда: ? 
Структура: ? [Аргумент] 
Отвечает за вывод подробной информации о команде
В качестве аргумента используется нужная команда
```

```
Команда: start
Структура: start [Аргумент] 
Отвечает за запуск задачи
Аргументы:
--name(-n) [Параметр]: указание названия задачи
```


```
Команда: stop
Структура: stop 
Отвечает за остановку текущей задачи
```


```
Команда: help
Структура: help 
Отвечает за вывод информации о доступных командах
```


```
Команда:  version
Структура: version
Отвечает за вывод текущей версии приложения
```


```
Команда: developer
Структура: developer 
Отвечает за вывод информации о разработчике
```


```
Команда: cancel
Структура: cancel
Отвечает за отмену задачи
```


```
Команда: tasks
Структура: tasks [Аргументы] 
Отвечает за управление задачами
Аргументы: 
--current(-c) : указывает на вывод информации о текущей задаче 
--date(-d) [Параметр]: указывает дату для вывода задача в определенный день
--sync(-s) : производит синхронизацию задач и поставленных целей
```


```
Команда: settings
Структура: settings [Аргумент] 
Отвечает за настройку программы
Аргументы: 
--username(-u) [Параметр]: смена пользовательского имени
```


```
Команда: targets
Структура: targets [Аргумент]
Отвечает за управление задачами
Аргументы: 
--date(-d) [Параметр]: указывает на дату для поиска задач
--today(-t) [Параметр]: указывает на сегодняшнюю дату
--yesterday(-y) [Параметр]: указывает на вчерашнюю дату
```


```
Команда: targets-add
Структура: targets-add [Аргументы] 
Отвечает за добавление задач
Аргументы: 
--name(-n) [Параметр]: указывает на название задачи
--date(-d) [Параметр]: указывает на дату задачи
--minutes(-m) [Параметр]: указывает на продолжительность задачи 
```


```
Команда: targets-del
Структура: target-del [Аргумент] 
Отвечает за управление задачами
Аргументы: 
--date(-d) [Параметр]: указывает на дату задачи
--name(-n) [Параметр]: указывает на название задачи
```
